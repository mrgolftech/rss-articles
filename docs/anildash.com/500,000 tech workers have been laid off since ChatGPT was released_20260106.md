# 500,000 tech workers have been laid off since ChatGPT was released

One of the key points I repeated when [talking about the state of the tech industry yesterday](https://www.anildash.com/2026/01/05/a-tech-career-in-2026/) was the salient fact that _half a million tech workers have been laid off since ChatGPT was released in late 2022_. The only advantage a large organization can have over a small one, other than sheer efficiencies of scale, is if it has a cognitive surplus that it can tap into.
