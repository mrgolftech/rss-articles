[Google Pinpoint](https://journaliststudio.google.com/pinpoint/about/) is Google product. Only use Pinpoint for datasets when you don't mind sharing a copy of your data with Google. It's great for completely public documents like these, but I recommend against using it for private data. Instead, use a tool like Aleph for that.
